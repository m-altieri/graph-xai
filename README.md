# Requirements

The software was tested on the following environment and might not work for
different versions:

- Python 3.9.16
- TensorFlow 2.12.0
- Torch 2.2.1+cu121
- CUDA 12.4
- Ubuntu 22.04.4 LTS

Download the data:

```
wget https://zenodo.org/records/13314559/files/data.zip -O data.zip
unzip data.zip -d . && rm data.zip
```
